# Privacy Policy

Sky Auto Piano does not collect, store, sell, or transmit personal data.

The application runs locally on the user's device. It does not include telemetry, analytics, advertising, account tracking, or crash reporting services.

Sky Auto Piano uses keyboard automation only to play music notes in Sky: Children of the Light. It does not log keystrokes, capture credentials, monitor user activity, inject into other applications, or send user input to external servers.

Official releases are distributed through GitHub Releases.
